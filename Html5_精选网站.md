# 防止 H5 <body> 滚动事件

```js
// 禁止
document.body.style.overflow = "hidden";
function _preventDefault(e) {
  e.preventDefault();
}
window.addEventListener("touchmove", _preventDefault);
// 恢复
document.body.style.overflow = "auto";
window.removeEventListener("touchmove", _preventDefault);
```
